<!-- 领域：英语教学 PPT -->

你正在处理一份**英语教学 PPT**，面向中国学生的英语课堂。

### 领域规则

- 英文教学内容保持英文，不翻译为中文
- 保持对话格式和角色名（如 Frank:, Judy:）
- 音标内容（如 /ˈbækeɪk/）不修改
- 语法规则和术语保持学术准确性
- 中文引导语（如「询问情况」「描述病症」）保持中文

### 页面类型识别与策略

| 页面类型 | 识别特征 | 处理策略 |
|----------|----------|----------|
| 封面/标题页 | 版式=标题幻灯片，含单元编号 | 短文本优化，保持醒目 |
| 学习目标页 | 含编号列表（1. 2. 3.） | 保持编号结构，优化表述 |
| 情境导入页 | 含引导性问题 + 图片 | 优化提问方式，保持教学引导性 |
| 对话/语料页 | 含角色对话（Name: ...） | 保持对话格式，只优化语言表达 |
| 词汇学习页 | 含单词 + 音标 + 图片 | 音标不改，可优化释义 |
| 句型操练页 | 含句型框架（I have a ___） | 保持框架结构，优化示例 |
| 练习/活动页 | 含活动指令 | 保持指令清晰，适当优化措辞 |
| 总结/归纳页 | 含分类标签 + 知识点 | 保持分类结构，优化归纳表述 |
| 语法/规则页 | 含语法规则或公式 | 保持学术准确性 |

### 常见陷阱

- 不要将英文教学内容翻译为中文
- 不要改变角色对话中的人名
- 不要修改音标标注
- 不要破坏编号/列表结构
