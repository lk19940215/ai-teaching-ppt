<!-- 操作：融合 (fuse) — 你必须创造全新的融合页面 -->

你收到了多个幻灯片页面的内容。你的任务是**真正融合它们**，而不是简单修改某一页。

### 核心要求

1. **必须创建至少一个新页面**（`is_new: true`），将多页内容融合为新的、有组织的内容
2. **不要只修改已有页面** — 融合的核心价值在于产出全新的整合内容
3. 新页面应提炼所有输入页面的精华，重新组织结构，而非简单拼接
4. 如果某些已有页面的内容被融合到新页面中了，可以修改已有页面（删减重复部分）

### 融合策略

- 提取每页的**核心教学目标**和**关键知识点**
- 找出**互补关系**（如：一页提供概念，另一页提供例句 → 融合为概念+例句页）
- 找出**重复内容**并精简
- 按逻辑重组（如：总览 → 重点 → 练习）
- 每个新页面应有明确的标题和结构化的正文

### 输出要求

- `slides` 数组中**必须**包含至少一个 `is_new: true` 的新页面
- 新页面的 `title`：简短有力，概括融合主题（10字以内为佳）
- 新页面的 `body_texts`：每条是一个**独立内容板块**，用标题行 + 换行 + 内容组织：
  ```
  "板块标题:\n• 要点1\n• 要点2\n• 要点3"
  ```
  例如：`["词汇学习:\n• headache 头痛\n• toothache 牙痛", "句型练习:\n• What's the matter?\n• You should..."]`
- 每页 body_texts 控制在 3-5 条，每条不超过 5 行，**避免内容过多导致溢出**
- 可以同时为新页面添加 `animation_hints`（如 fade、fly_in）
- `layout_hint` 推荐用 `title_and_content`
- **不要修改已有页面**，聚焦于创建高质量的新融合页面
