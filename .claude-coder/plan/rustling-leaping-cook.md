# AI 教学 PPT 生成器 - 首页重设计方案

## Context

当前首页实现过于简洁，缺乏视觉层次和吸引力。本次重设计旨在：
- 提升视觉美感，打造现代化AI产品形象
- 改善用户体验，清晰展示产品价值和使用流程
- 增加功能入口，让用户快速找到需要的功能

## 设计风格

现代简约 + 科技感渐变，适合AI教育产品定位。

## 实现步骤

### Step 1: 重写首页 page.tsx

替换现有内容为新的六区块布局：

1. **Hero区**: 渐变背景、动态光晕、徽章标签、渐变标题、双CTA按钮
2. **流程区**: 三步骤卡片，使用 `stagger-item` 动画依次显示
3. **特性区**: Bento Grid布局，大卡片突出AI能力
4. **统计区**: 4列数据展示（学科数、年级覆盖、输入方式、兼容性）
5. **入口区**: 4个快速导航卡片，悬浮时图标放大
6. **CTA区**: 渐变背景 + 突出按钮

### Step 2: 使用的动画类

| 区块 | 动画类 |
|-----|--------|
| Hero徽章 | `animate-fade-in-down` |
| Hero标题/副标题/按钮 | `animate-fade-in-up` |
| 流程卡片 | `stagger-item` |
| 功能卡片 | `stagger-item` + `card-hover` |
| 背景光晕 | `animate-pulse-slow` |

### Step 3: 响应式设计

- 移动端 (<640px): 单列布局，按钮堆叠，统计2列
- 平板 (640-768px): 两列布局
- 桌面 (>768px): 完整三列布局，显示流程连接线

## 验证方式

1. 运行开发服务器: `cd frontend && npm run dev`
2. 访问 http://localhost:3000 查看首页效果
3. 测试不同屏幕尺寸的响应式表现
4. 验证所有链接跳转正常（/upload, /merge, /history, /settings）
