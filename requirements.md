# AI 教学 PPT 生成器

## 项目概述

做一个 Web 应用：教师上传教材内容（拍照图片 / PDF 电子书 / 直接粘贴文字），选择年级和学科后，AI 自动生成互动性强、美观、知识点丰富的教学 PPT。生成的 PPT 为标准 .pptx 格式，可在 WPS Office 中直接打开和二次编辑。

目标用户：小学（1-6年级）和初中（7-9年级）的教师。
学科范围：通用（所有学科），英语学科做特别强化。

## 功能需求

### 核心流程

- [ ] 功能 1：教材内容输入 — 支持三种方式上传教材内容
  - 拍照/图片上传：支持 JPG、PNG 格式，支持多张图片同时上传，通过 OCR 识别教材文字（中英文混合）
  - PDF 上传：支持 PDF 电子书上传，自动提取文字和图片内容
  - 文字输入：提供文本框，教师可直接粘贴课本内容或手动输入教学要点
- [ ] 功能 2：教学配置面板 — 教师选择生成参数
  - 年级选择：小学1-6年级、初中7-9年级
  - 学科选择：语文、数学、英语、科学、物理、化学、生物、历史、政治、地理、通用
  - 章节信息：可选填写章节名称（如"第三章 分数的加减法"）
  - PPT 风格选择：活泼趣味（适合低年级）、简约清晰（适合高年级）、学科主题（根据学科自动配色）
  - 幻灯片数量：可选范围 8-30 页，默认 15 页
- [ ] 功能 3：AI 内容生成 — 调用 LLM 分析教材内容并生成结构化的 PPT 内容
  - 根据教材原文提炼知识点、重难点
  - 根据年级调整内容深度和表达方式
  - 生成互动环节（课堂提问、小组讨论题、趣味问答）
  - 生成记忆口诀或助记方法
  - 自动生成课堂练习题
- [ ] 功能 4：PPT 文件生成 — 使用 python-pptx 生成标准 .pptx 文件
  - 页面类型包含：封面页、目录页、知识点讲解页、互动问答页、课堂练习页、总结回顾页
  - 自动排版：合理的字号、行距、配色
  - 内置多套教学 PPT 模板
  - 生成的 .pptx 文件可在 WPS Office 中正常打开和编辑
- [ ] 功能 5：PPT 预览与下载
  - 生成后在页面上展示 PPT 每页的缩略图预览
  - 支持一键下载 .pptx 文件
  - 支持重新生成（不满意时可调整参数后重试）

### 英语学科特殊功能

- [ ] 功能 6：英语教学 PPT 增强
  - 自动生成单词卡片页：单词 + 音标 + 中文释义 + 英文例句
  - 语法结构图解页：用表格或图示展示语法规则
  - 情景对话页：生成贴合课文主题的对话示例
  - 课文要点分析页：段落大意、关键句型、常用表达
  - 课堂互动页：填空、选择、连线等练习形式

### 按年级分层设计

- [ ] 功能 7：年级自适应 PPT 风格
  - 小学（1-3年级）：特大字号（不小于28pt）、鲜艳配色、大量留白、趣味化表达、拼音标注（语文）
  - 小学（4-6年级）：较大字号（不小于24pt）、明快配色、图文并茂、设置互动游戏式问答
  - 初中（7-9年级）：标准字号（不小于20pt）、稳重配色、知识体系化、思维导图、重点标注、例题解析

### 辅助功能

- [ ] 功能 8：生成历史记录 — 保存用户之前生成的 PPT 记录
  - 展示生成时间、教材主题、年级学科
  - 可重新下载已生成的 PPT
  - 可基于历史记录快速重新生成
- [ ] 功能 9：LLM 服务商切换 — 支持在设置页面配置和切换 AI 服务商
  - 支持 DeepSeek、OpenAI、Claude、智谱 GLM
  - 用户填入自己的 API Key
  - 可设置默认服务商

## 技术约束

- 后端：Python 3.11+ FastAPI
- 前端：Next.js 14 + TypeScript + Tailwind CSS + shadcn/ui
- 前后端分离部署，前端通过 API 调用后端
- OCR：PaddleOCR（中文识别准确率远高于 Tesseract）
- PDF 解析：PyMuPDF (fitz)
- PPT 生成：python-pptx（生成标准 .pptx，兼容 WPS）
- LLM 调用：统一抽象接口，底层支持 OpenAI 兼容格式（DeepSeek / OpenAI / Claude / GLM 都可通过 OpenAI SDK 调用）
- 数据库：SQLite（轻量级，存储生成历史和用户配置）
- 文件存储：本地文件系统存储上传的教材和生成的 PPT
- 包管理：后端用 pip + requirements.txt，前端用 pnpm

## 样式与设计

- 整体风格：简约、现代、教育感，参考 Apple 教育官网的清爽感
- 配色：主色调 #4F46E5（靛蓝），辅助色 #10B981（翠绿），背景白色
- 移动端适配：是（教师可能用手机拍照上传）
- CSS 框架：Tailwind CSS + shadcn/ui 组件库
- 上传区域：大面积拖拽区，支持点击上传，显示上传进度
- 加载状态：生成 PPT 时展示进度条和趣味等待提示（如"AI 正在备课中..."）

## 其他要求

- 界面语言：中文
- 单次生成 PPT 时间控制在 30 秒以内（不含 LLM 响应等待）
- 上传的图片和 PDF 文件大小限制：单文件不超过 20MB
- 生成的 PPT 文件需要在 WPS Office 2019 及以上版本中正常显示
- 前后端都需要提供 Docker 部署支持（Dockerfile）
- 后端提供 OpenAPI 文档（FastAPI 自带 /docs 页面）
- 错误处理友好：OCR 识别失败、LLM 调用超时等异常情况给用户明确提示
