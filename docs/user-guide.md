# PPT 智能合并工具 - 用户指南

## 快速开始

### 1. 配置 AI 模型

首次使用需要配置 AI 模型的 API Key：

1. 访问 http://localhost:3000/settings
2. 选择一个 AI 服务商（推荐 DeepSeek，性价比高）
3. 输入 API Key
4. 点击「测试连接」验证配置
5. 点击「保存」

**支持的 AI 服务商：**

| 服务商 | 特点 |
|---|---|
| DeepSeek | 性价比高，中文效果好 |
| OpenAI | GPT 系列，功能强大 |
| Claude | 推理能力强，适合复杂任务 |
| 智谱 GLM | 国产大模型，中文优化 |

### 2. 上传 PPT

1. 访问 http://localhost:3000/merge
2. 点击上传区域或拖拽 PPTX 文件
3. 依次上传两个 PPT 文件
4. 等待解析完成，页面会显示所有幻灯片预览

**注意：**
- 支持的格式：.pptx（不支持 .ppt 旧格式）
- 文件大小：建议不超过 50MB
- 页数限制：建议单个 PPT 不超过 100 页

### 3. 处理幻灯片

选中一个幻灯片后，可以进行以下操作：

| 操作 | 功能 | 适用场景 |
|---|---|---|
| **润色** | 优化文字表达，保持原意 | 内容粗糙，需要精炼 |
| **扩展** | 补充更多细节和示例 | 内容单薄，需要丰富 |
| **改写** | 以不同风格重写内容 | 需要调整表达风格 |
| **提取** | 提取知识点和关键信息 | 需要总结要点 |

**操作步骤：**
1. 点击选中一个幻灯片
2. 选择处理动作（润色/扩展/改写/提取）
3. 可选：输入自定义提示词
4. 点击「开始处理」
5. 等待 AI 处理完成
6. 查看处理结果，可在版本间切换

### 4. 合并页面

将多个幻灯片合并为一张新幻灯片：

1. 按住 Ctrl/Cmd 多选幻灯片
2. 点击「合并选中」
3. 等待 AI 融合处理
4. 合并结果会出现在幻灯片池中

**合并策略：**
- AI 会分析各页面内容
- 提取共同主题和关键信息
- 生成结构化的融合内容

### 5. 生成最终 PPT

1. 点击幻灯片右侧的「添加」按钮
2. 幻灯片会出现在底部「最终选择」栏
3. 拖拽调整顺序
4. 点击「生成最终 PPT」
5. 下载生成的 .pptx 文件

**生成的 PPT 特点：**
- 标准 .pptx 格式
- 支持 WPS 和 PowerPoint 编辑
- 保留 AI 处理后的内容结构

## 界面说明

### 三栏布局

```
┌──────────────┬──────────────┬──────────────┐
│   幻灯片池   │    预览区    │   操作面板   │
│              │              │              │
│  ┌──┐┌──┐   │   ┌────┐     │  选择动作    │
│  │1 ││2 │   │   │预览│     │  输入提示词  │
│  └──┘└──┘   │   └────┘     │  开始处理    │
│  ┌──┐┌──┐   │              │              │
│  │3 ││4 │   │   版本切换   │              │
│  └──┘└──┘   │              │              │
└──────────────┴──────────────┴──────────────┘
```

- **左侧**：幻灯片池，显示所有可用幻灯片
- **中间**：预览区，显示选中幻灯片的详细内容
- **右侧**：操作面板，处理动作和提示词输入

### 底部选择栏

显示已选择要放入最终 PPT 的幻灯片，支持拖拽排序。

## 常见问题

### Q: 处理超时怎么办？

A: AI 处理可能需要 30-60 秒。如果超时：
1. 检查网络连接
2. 确认 API Key 有效
3. 刷新页面重试

### Q: 生成的 PPT 内容不理想？

A: 可以尝试：
1. 使用「自定义提示词」给出更具体的要求
2. 选择不同的处理动作
3. 切换其他 AI 服务商

### Q: 上传的 PPT 解析失败？

A: 可能原因：
1. 文件格式不支持（需要 .pptx）
2. 文件损坏
3. 包含特殊元素（如复杂图表、3D 模型）

### Q: 如何查看历史处理记录？

A: 每个幻灯片支持多版本。点击版本标签可切换查看不同处理结果。
