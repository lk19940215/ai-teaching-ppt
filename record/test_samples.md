## 测试示例 1：小学数学 - 分数的认识

**年级**：小学三年级
**学科**：数学
**风格**：活泼趣味
**幻灯片数量**：12 页

**教材内容**：

```
分数的认识

一、什么是分数
分数是用来表示整体的一部分的数。
把一个苹果平均分成 4 份，每份就是这个苹果的四分之一，写作 1/4。

二、分数的组成
分数由分子、分母和分数线组成。
例如：3/4
- 3 是分子，表示取的份数
- 4 是分母，表示平均分的总份数
- 中间的横线是分数线

三、分数的读法
1/2 读作：二分之一
3/4 读作：四分之三
5/8 读作：八分之五

四、分数的大小比较
分母相同，分子大的分数大。
例如：3/5 > 2/5

分子相同，分母小的分数大。
例如：1/2 > 1/3

五、生活中的分数
- 一块蛋糕切成 8 块，吃了 3 块，就是吃了 3/8
- 一小时的四分之一是 15 分钟
- 一米的一半是 1/2 米
```

## 测试示例 2：初中语文 - 古诗词鉴赏

**年级**：初中一年级
**学科**：语文
**风格**：简约清晰
**幻灯片数量**：15 页

**教材内容**：

```
《静夜思》鉴赏 - 李白

一、作者介绍
李白（701-762），字太白，号青莲居士。
唐代伟大的浪漫主义诗人，被后人誉为"诗仙"。
代表作有《望庐山瀑布》《将进酒》《早发白帝城》等。

二、原文
床前明月光，
疑是地上霜。
举头望明月，
低头思故乡。

三、词语解释
- 疑：怀疑，以为
- 举头：抬头
- 思：思念，想念

四、诗歌大意
明亮的月光洒在床前，
让人怀疑是地上结了一层白霜。
抬头仰望天上的明月，
不禁低头思念远方的故乡。

五、写作背景
这首诗写于唐玄宗开元十四年（726 年），
当时李白 26 岁，离开故乡在扬州游玩。
在一个寂静的夜晚，看到明月，
触发了对故乡的深深思念。

六、艺术特色
1. 语言朴实自然，如话家常
2. 意境深远，耐人寻味
3. 运用对比手法：举头 - 低头，望 - 思
4. 以月为媒介，寄托思乡之情

七、思想感情
表达了诗人对故乡的深切思念之情，
以及漂泊在外的孤独感。

八、名句赏析
"举头望明月，低头思故乡"
- 动作描写细腻：举头、低头
- 情感真挚：思故乡
- 成为千古传诵的名句
```

## 测试示例 3：小学英语 - 颜色单词

**年级**：小学三年级
**学科**：英语
**风格**：活泼趣味
**幻灯片数量**：10 页

**教材内容**：

```
Unit 3 Colors - 颜色单词学习

一、核心单词
1. red - 红色 🍎
2. blue - 蓝色 🌊
3. yellow - 黄色 🌻
4. green - 绿色 🌿
5. orange - 橙色 🍊
6. purple - 紫色 🍇
7. pink - 粉色 🌸
8. brown - 棕色 🪵
9. black - 黑色 ⚫
10. white - 白色 ⚪

二、重点句型
1. What color is it? - 它是什么颜色？
2. It's red. - 它是红色的。
3. I like blue. - 我喜欢蓝色。
4. The apple is red. - 苹果是红色的。

三、对话练习
A: What's your favorite color?
B: My favorite color is green.

A: Look at the rainbow!
B: Wow! So many colors!

四、小练习
1. 苹果是什么颜色？The apple is ___.
2. 天空是什么颜色？The sky is ___.
3. 香蕉是什么颜色？The banana is ___.

五、趣味知识
- 彩虹有七种颜色：红橙黄绿青蓝紫
- 英文：Red, Orange, Yellow, Green, Blue, Indigo, Violet
- 记忆口诀：ROY G BIV（人名）

六、课后作业
1. 抄写单词 each 3 遍
2. 用英文说出家中物品的颜色
3. 画一幅彩色的画，并用英文标注颜色
```

## 测试示例 4：初中物理 - 牛顿第一定律

**年级**：初中二年级
**学科**：物理
**风格**：简约清晰
**幻灯片数量**：14 页

**教材内容**：

```
牛顿第一定律（惯性定律）

一、导入思考
为什么急刹车时，身体会向前倾？
为什么拍打衣服可以除去灰尘？

二、历史回顾
亚里士多德的观点：
- 力是维持物体运动的原因
- 物体不受力就会停止运动

伽利略的理想实验：
- 小球从斜面滚下，会在水平面上继续运动
- 水平面越光滑，小球运动得越远
- 推理：如果没有阻力，小球将永远运动下去

三、牛顿第一定律
一切物体在没有受到外力作用时，
总保持静止状态或匀速直线运动状态。

公式表达：F合 = 0 时，v = 常量

四、惯性概念
物体保持原来运动状态不变的性质叫做惯性。
- 静止的物体倾向于保持静止
- 运动的物体倾向于保持运动

五、惯性的应用
1. 锤头松了，把锤柄在地上撞击几下
2. 跳远运动员要助跑
3. 汽车安全带的作用

六、惯性的防止
1. 汽车安全带和安全气囊
2. 交通规则禁止超载
3. 易碎物品包装填充缓冲材料

七、影响惯性的因素
惯性大小只与物体的质量有关。
- 质量越大，惯性越大
- 质量越小，惯性越小

八、典型例题
例 1：为什么公交车启动时，乘客会向后仰？
答：启动前，乘客和车都静止。启动时，
脚随车向前运动，而上身由于惯性保持静止，
所以会向后仰。

例 2：百米赛跑到达终点后，运动员为什么不能立即停下来？
答：运动员具有惯性，要保持原来的运动状态，
所以不能立即停下来。

九、课堂小结
1. 牛顿第一定律的内容
2. 惯性的定义
3. 惯性大小的决定因素：质量
4. 惯性在生活中的应用和防止
```

## 使用说明

1. 复制任意一个测试示例的内容
2. 访问 http://localhost:3000/upload
3. 选择"文字输入"方式，粘贴内容
4. 选择对应的年级、学科、风格
5. 点击"生成教学 PPT"按钮
6. 预览并下载生成的 PPT 文件
